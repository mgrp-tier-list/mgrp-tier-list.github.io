This is like a Smash Tier List Maker, but with Magical Girl Raising Project instead.

Source code can be found here and was forked from here.

Portraits are sourced from the MGRP Wikia, and property of Asari Endou, Marui-no, and Takarajimasha.

Clicking on the "Dock tier list" button will stick the tier list to the top of the window. This can be useful if you have a large number of characters on-screen and don't want to scroll up and down as you drag them into your list. Clicking the button again will return the list to its original position.

To take a screenshot, you can click the "Hide controls" button to make the list look cleaner. Then, use a screenshot tool such as Snipping Tool on Windows to take a screenshot.

There is a gear next to each tier in the tierlist. If you click on it, you can change the color of the tier label, add rows, delete the row, or remove all of the characters currently in the row.

Next to the gear are 2 arrows, pointed up and down. Clicking these will let you move a tier up and down a level.

You can change the name of the tier. Click on the tier label, you can type in there. If this doesn't work, you can change it in the textbox in the corresponding settings menu.

If you want to completely start over, refresh the page or click the "Reset" button at the top of the page.

For any issues, add it to the list on the repository page or message me if you know where to find me.
